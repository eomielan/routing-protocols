Compute the forwarding table for a given source node.

This function computes the forwarding table for a given source node in a distance vector graph. It uses the table of distance vectors for the source node to determine the next hop and path cost for each destination node, and ignores any unreachable nodes.

Parameters
graphThe distance vector graph.
source_nodeThe source node for which to compute the forwarding table.
Returns
A list of tuples, where each tuple contains the destination node, the next hop, and the path cost.

Write the forwarding tables to an output file.

This function writes the forwarding tables for all nodes in a distance vector graph to an output file. Each entry is separated by a new line. The format for each entry in the forwarding table is as follows:

<destination> <next_hop> <path_cost>

Write the messages to an output file.

This function writes the messages from a message file to an output file. For each message, it determines the path and path cost to the destination node using the distance vector of the source node, and writes the message along with the path and path cost to the output file.

If the destination node is reachable from the source node, the output will be formatted as:

from <src_id> to <dest_id> cost <pathCost> hops <hop1> <hop2> <...> message <message>

If the destination node is unreachable from the source node, the output will be formatted as:

from <src_id> to <dest_id> cost infinite hops unreachable message <message>
Parameters
graphThe distance vector graph.
msg_fileThe message file to read the messages from.
output_fileThe output file to write the messages to.
