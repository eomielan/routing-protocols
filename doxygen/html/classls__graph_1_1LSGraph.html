Detailed Description

A class representing a Link State Graph.

This class is used to represent a Link State Graph, which is a type of graph used in network routing protocols.

Get the shortest path between two nodes using Dijkstra's algorithm.

Parameters
selfThe object pointer.
source(int): The starting node.
dest(int): The ending node.
Returns
List[int]: The shortest path between the two nodes, or an empty list if no path exists. If multiple shortest paths exist, the path with the lowest node ID prior to the destination node is returned. If there is a tie, the algorithm keeps checking the previous node until the tie is broken.
